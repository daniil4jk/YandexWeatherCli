# YandexWeatherCli

CLI-утилита для прогноза погоды через Yandex Weather API. Возвращает машиночитаемый вывод.

## Requirements

- **Java 17+** (проверить: `java -version`)
- Файл `api-key.txt` с ключом Yandex Weather API (рядом с JAR)
- Опционально: `places.json` со списком городов

## Setup

1. Убедись, что Java 17+ доступна:
   ```bash
   java -version  # должно быть 17, 21 или новее
   ```
   Если нет — установи через пакетный менеджер:
   - Ubuntu/Debian: `sudo apt install openjdk-21-jre`
   - RedHat/Fedora: `sudo yum install openjdk-21-jdk-headless`
   - Windows: скачай с https://adoptium.net/

2. Положи `api-key.txt` рядом с JAR:
   ```
   ваш-ключ-яндекс-погоды
   ```

3. (Опционально) Отредактируй `places.json` — добавь города и места пользователя с алиасами имён

## Usage

```bash
java -jar yweather.jar --place Екатеринбург
java -jar yweather.jar --lat 56.83 --lon 60.60 --hours 0-12
```

### Флаги

| Флаг | Описание |
|------|----------|
| `--place <name>` | Город из `places.json` |
| `--lat / --lon` | Координаты |
| `--compact` | Машиночитаемый вывод (для агента) |
| `--hours <N>` | Час (`5`) или диапазон (`0-12`) |
| `--fields` | Какие поля выводить |

## Доступные поля в --fields

| Поле | Тип | Описание | Пример |
|------|-----|----------|--------|
| `hour` | число | Час от 0 до 23 | `14` |
| `temp` | °C | Температура воздуха | `+18`, `-5` |
| `feels_like` | °C | Ощущаемая температура | `+16` |
| `condition` | строка | Погодное явление (маппится на русский) | `Ясно`, `Дождь`, `Гроза с градом` |
| `prec_strength` | число | Интенсивность осадков (0 = нет, >0 = мм/ч) | `0.0`, `1.5` |
| `prec_type` | enum | Тип осадков: `none` / `rain` / `snow` | `rain` |
| `wind_speed` | м/с | Скорость ветра | `3.2` |
| `wind_gust` | м/с | Порывы ветра | `7.1` |
| `humidity` | % | Влажность | `65%` |
| `wind_dir` | строка | Направление ветра (румбы) | `С`, `ЮЗ`, `СВ` |

### Примечания к полям

- **`prec_type`** — внутри хранится как int: `0` → без осадков, `1` → дождь, `2` → снег
- **`wind_dir`** — маппинг румбов Яндекс.Погоды: `n→С, ne→СВ, e→В, se→ЮВ, s→Ю, sw→ЮЗ, w→З, nw→СЗ`
- **`condition`** — все коды погодных явлений маппятся на русские названия (ясно, дождь, гроза и т.д.)
- **`temp`** — всегда с явным знаком (`%+.0f`)
- **`prec_strength`** — сырое значение из JSON; 0 = без осадков, >0 = интенсивность в мм/ч

Строка с pipe-разделителями:

```
Екатеринбург | 2026-06-14 | +18°C | облачно
```

Если часов несколько — каждая строка с префиксом.

## places.json

```json
[
  {
    "names":
    ["Москва", "Moscow"],
    "lat": 55.755864,
    "lon": 37.617698
  },
  {
    "names":
    ["Лондон", "London"],
    "lat": 51.507351,
    "lon": -0.127696
  }
]
```

Поле `names` — массив имён (русское, английское), по любому из них сработает `--place`.
